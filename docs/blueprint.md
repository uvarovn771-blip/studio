# **App Name**: Dark Ascent Landing

## Core Features:

- Hero Section: Eye-catching hero section with course title, concise description, and a prominent call-to-action button.
- Course Benefits: Dedicated section highlighting the key benefits of the course, using concise text and engaging icons.
- About the Author: Showcase the author's expertise with a professional headshot, brief biography, and relevant credentials.
- FAQ Section: Interactive FAQ section with a clean, expandable/collapsible design to address common questions and concerns.
- Buy Button: Fixed "Buy" button which remains visible at the bottom of the screen on mobile.

## Style Guidelines:

- Primary color: Deep Indigo (#4F46E5) for a modern and sophisticated feel.
- Background color: Dark Gray (#1F2937) to provide a clean, modern dark theme.
- Accent color: Teal (#2DD4BF) for highlights and call-to-action elements.
- Font: 'Inter', a grotesque-style sans-serif suitable for headlines or body text.
- Responsive layout using Tailwind CSS grid and flexbox to ensure optimal viewing on all devices.
- Use simple, consistent icons from a library like Feather or Font Awesome to illustrate course benefits.
- Subtle animations like fade-ins on scroll to enhance user engagement without being distracting.
